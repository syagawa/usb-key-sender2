# Keys

## Modifier keys

* Shift:`{"mod": "SHIFT"}`
    * (Example) Shift Key + A: `{"mod": "SHIFT", "key": "a"}`
* Control:`{"mod": "CTRL"}`
    * (Example) Ctrl Key + C : `{"mod": "CTRL", "key": "c"}`
* Alt:`{"mod": "ALT"}`
    * (Example) Alt Key + L: `{"mod": "ALT", "key": "l"}`
* GUI Key (Windows Key / Command Key on Mac): `{"mod": "GUI"}`
    * (Example) Windows Key + D: `{"mod": "GUI", "key": "d"}`
* Multiple Modifiers Support Chain keys using | to press them simultaneously:: `{"mod": "SHIFT|CTRL|ALT", "key": "s"}`
